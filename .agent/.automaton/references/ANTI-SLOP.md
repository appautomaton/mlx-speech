# Anti-Slop

Prose patterns to flag in any Automaton artifact. Flag each hit unless it is quoted source text, matches the user's approved voice, or is intentionally justified.

## Patterns

| Pattern | Check for |
| --- | --- |
| Significance inflation | "pivotal," "crucial," "vital," "testament," "key moment," or inflated stakes not supported by evidence. |
| Promotional language | "vibrant," "groundbreaking," "nestled," "stunning," "breathtaking," or press-release adjectives. |
| Superficial `-ing` padding | "highlighting," "underscoring," "emphasizing," or "showcasing" as sentence filler. |
| Vague attribution | "experts say," "industry observers note," "many believe," or unattributed authority claims. |
| Em-dash overuse | Dash-heavy connective tissue. Use commas, periods, semicolons, or colons instead. |
| Forced rule of three | Ideas shaped into three-part lists for rhythm rather than meaning. |
| Sycophantic artifacts | "Great question!", "Certainly!", "I hope this helps!", or assistant residue. |
| Generic conclusions | "The future looks bright," "exciting times ahead," or endings that avoid a concrete claim. |
| Copula padding | "serves as," "stands as," "boasts," where "is," "are," or "has" is clearer. |
| Signposting | "Let's dive in," "Here's what you need to know," or preambles that delay content. |
| Unsupported specificity | Precise numbers, rankings, dates, or claims presented without source support. |
