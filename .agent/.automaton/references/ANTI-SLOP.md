# Anti-Slop Taxonomy

Use this reference for content-mode drafting, framing, and verification. It defines concrete prose patterns to remove or justify; it is not a general writing-quality guide.

## Rule

Flag each hit unless it is quoted source text, required by the user's approved voice sample, or intentionally justified for the artifact.

## Patterns

| Pattern | Check for |
| --- | --- |
| Significance inflation | "pivotal," "crucial," "vital," "testament," "key moment," or inflated stakes not supported by the source. |
| Promotional language | "vibrant," "groundbreaking," "nestled," "stunning," "breathtaking," or press-release adjectives. |
| Superficial `-ing` analysis | "highlighting," "underscoring," "emphasizing," or "showcasing" used as sentence padding. |
| Vague attribution | "experts say," "industry observers note," "many believe," or unattributed authority claims. |
| Em-dash overuse | Dash-heavy connective tissue that can be rewritten with commas, periods, or parentheses. |
| Forced rule of three | Ideas shaped into three-part lists or conclusions for rhythm rather than meaning. |
| Sycophantic artifacts | "Great question!", "Certainly!", "I hope this helps!", or similar assistant residue. |
| Generic conclusions | "The future looks bright," "exciting times ahead," or endings that avoid a concrete claim or action. |
| Copula padding | "serves as," "stands as," "boasts," or similar evasions where "is," "are," or "has" is clearer. |
| Signposting | "Let's dive in," "Here's what you need to know," or preambles that delay the content. |
| Unsupported specificity | Precise examples, numbers, rankings, dates, or claims presented without allowed source support. |

## Use

- Intake: name the specific pattern when asking for content anti-goals.
- Framing: scan SPEC.md itself so the instructions do not model the slop they reject.
- Execution: remove the pattern directly or record why it stays.
- Verification: each unquoted, unjustified hit is FAIL or PARTIAL evidence.
