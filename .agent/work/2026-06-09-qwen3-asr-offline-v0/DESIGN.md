## Architecture Approach
Qwen3-ASR lands as a new isolated ASR family, not as a retrofit of Cohere ASR or Granite Speech ASR. The runtime surface stays consistent with existing ASR adapters, while model-specific code lives under a dedicated Qwen package.

## Module Layout
- `src/mlx_speech/models/qwen3_asr/config.py`: parse `model_type=qwen3_asr`, thinker/audio/text configs, token IDs, and supported languages from `config.json`.
- `src/mlx_speech/models/qwen3_asr/tokenizer.py`: load Qwen tokenizer assets with `tokenizers`, resolve audio token strings/IDs from config and tokenizer files.
- `src/mlx_speech/models/qwen3_asr/feature_extraction.py`: pure NumPy/MLX WhisperFeatureExtractor-parity audio frontend.
- `src/mlx_speech/models/qwen3_asr/processor.py`: build explicit token-id prompts, expand audio placeholders, compute feature lengths, parse output language/text.
- `src/mlx_speech/models/qwen3_asr/audio_encoder.py`: MLX Conv2D + audio transformer encoder.
- `src/mlx_speech/models/qwen3_asr/text_decoder.py`: Qwen3 decoder with QK norm, GQA, RoPE, and KV cache.
- `src/mlx_speech/models/qwen3_asr/model.py`: audio feature extraction, placeholder mask checks, audio-text embedding fusion, prefill, and greedy decode.
- `src/mlx_speech/models/qwen3_asr/checkpoint.py`: sharded safetensors loading, key/layout sanitization, BF16 packaging, and alignment reports.
- `src/mlx_speech/generation/qwen3_asr.py`: runtime transcription wrapper and preflight checks.
- `src/mlx_speech/asr/_adapters/qwen3.py`: public ASR adapter.

## Checkpoint Policy
The downloaded upstream files under `models/qwen3_asr_1_7b/original/` are already BF16 safetensors. The v0 conversion target is `models/qwen3_asr_1_7b/mlx-bf16/`. Conversion must preserve BF16 except for tensor layout changes required by MLX modules, especially Conv2D weights. Low-bit quantization is out of scope.

## Language Semantics
Qwen's default is auto English/Chinese/mixed-language transcription. `None`, empty string, or `"auto"` must avoid the forced-language suffix. Explicit supported languages force text-only transcription. Cohere and Granite preserve their current English default by mapping omitted language to `"en"` internally.

## Prompt Strategy
Do not depend on `transformers.apply_chat_template` at runtime. Build the reference-equivalent prompt as explicit token IDs using local tokenizer assets, then expand Qwen audio placeholders to the computed audio feature length. The implementation must fail before generation if placeholder count and audio feature length differ.

## Runtime Boundary
v0 is single-pass offline inference with preflight input limits. Long-audio energy-boundary splitting, chunk merge, streaming, timestamps, forced alignment, automatic download, and quantization are deferred.
