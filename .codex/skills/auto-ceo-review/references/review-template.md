# Review Template: Product

Append exactly this format to `SPEC.md`:

```markdown
## Review: Product

- Verdict: <approved|approved_with_risks|needs_clarification|descoped>
- Strength: <one sentence>
- Concern: <one sentence>
- Action: <one sentence>
- De-scoped: <comma-separated list or "none">
```

## Rules

- Verdict must be one of the four approved values. No synonyms.
- Strength and Concern must each be exactly one sentence.
- Action must be a concrete next step, not a strategy.
- De-scoped must list specific items removed from scope, or "none."
- Do not add extra fields or commentary outside this format.
